02_제이쿼리 문서객체(dom)조작

dom(문서객체모델), html 문서안에서 모든 요소들과 내용들을 제이쿼리를 사용하여 글자를 변경하거나, 추가, 삭제, 속성값을 변경할 수 있는 것을 말한다.

http://oscarotero.com/jquery 웹사이트 참고. API에서 attributes(속성), manipulation(조작) 카테고리에서 메소드 확인 가능

문서내용을 변경하기 위한 메서드 종류

클래스 조작(removeClass, toggleClass, addClass, hasClass)

속성 조작 (attr)

내용 조작 (text, html)

  • $('대상').text(); - 텍스트 내용을 변경하기 위한 메서드
  • $('대상').html(); - html 태그를 포함하여 텍스트 내용 변경
  • $('대상').attr(); - 속성이나 속성값을 변경
  • $('대상').val(); - 선택요소 폼태그 value 값을 변경
  • $('대상').empty(); - 선택요소 내용 지우기
  • $('대상').remove(); - 선택요소를 지우기 (js에서는 classList.add)
  • $('대상').revoveClass(); - 클래스 제거 (js에서는 classList.remove)
  • $('대상').hasClass(); - 클래스 존재여부
  • $('대상').toggleClass(); - 클래스 적용, 제거
  • $('대상').appendTo(); - 부모내용 마지막 위치로 이동(자리바꾸기)
  • $('대상').prependTo(); - 부모내용 처음 위치로 이동(자리바꾸기)
  • $('A').insertAfter('B'); - B 요소 뒤에 A를 배치 (B - A)
  • $('A').insertBefore('B'); - B 요소 앞에 A를 배치 (A - B)
