01_제이쿼리_문서조작실습_카카오톡_채팅창

채팅창 입력양식을 삽입하여 사용자가 글내용 입력후 '전송'버튼 클릭시 글내용이 대화상자에 출력되게 하기

  • 채팅창 화면을 header, main, footer영역으로 구성한다
  • 상단 header 영역은 브라우저버튼, 사용자 프로필영역, 소개글 작성
  • main영역은 상대방과 사용자가 입력한 글 내용이 말풍선으로 출력
  • footer 영역은 사용자가 입력양식에 글을 입력하고 버튼 클릭시 변수에 담아서 내용을 출력하도록 한다
  • 사용자가 글 내용을 입력하지 않고, 버튼을 클릭하면 버튼이 비활성화(disabled)되도록 한다
  • 사용할 속성 메소드 : val /클래스 조작: addClass / 내용추가 : append
