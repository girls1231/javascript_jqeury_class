03제이쿼리 이벤트

웹문서에서 버튼을 클릭한다거나 마우스를 움직이거나 하는 모든 행위를 이벤트라고 한다. 그리고 이벤트 발생시 함수의 실행문이 수행되도록 이벤트를 지정하는 것을 '이벤트 핸들러'라고 한다

1. 이벤트의 종류

  • 마우스 이벤트
  • 키보드 이벤트
  • 포커스 이벤트
  • 폼양식 이벤트
  • 윈도우 이벤트

2. 이벤트를 등록하는 방법(작성법)

  • 단독이벤트 - $('이벤트 대상').이벤트 메서드(function(){실행할 내용});
  • 그룹이벤트 - $('이벤트 대상').on('이벤트 종류1','이벤트 종류2'.....,function(){실행할 내용});

3. 마우스 이벤트와 폼양식 이벤트

  1. click - 마우스로 대상을 클릭 할 때 이벤트 실행
  2. dbclick - 마우스로 대상을 더블클릭 할 때 이벤트 실행
  3. mouseenter - 마우스로 대상을 오버할 때 이벤트 실행(=mouseover)
  4. mouseleave - 마우스로 대상을 오버했다가 뺐을 때 이벤트 실행(=mouseout)
  5. hover - 마우스로 디상을 오버했을 때와 뺐을 대 둘다 각각 따로 이벤트 내용 실행
  6. focus - 마우스로 클릭하여 포커스가 발생되었을 대 이벤트 발생(양식에서)
  7. blur - 폼양식에서 포커스가 해제되었을 대 발생
  8. toggle - 대상 요소를 클릭시마다 지정된 함수를 호출하여 실행
  9. each - 개별(각각에 대해)처리하는 이벤트 함수
