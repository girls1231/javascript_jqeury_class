02 제이쿼리 선택자(selector)

제이쿼리에서는 css사용하던 선택자를 활용하여 서식을 적용할 수 있는데, 제이쿼리는 css보다 좀 더 세밀하게 선택할 수 있고, 그 개수도 많다.

http://oscarotero.com/jquery/

제이쿼리 선택자를 사요하여 서식적용하기

  • 아이디 선택자를 사용하여 글자색 변경하기
  • 클래스 선택자를 사용하여 배경색 변경하기
  • 태그 선택자를 사용하여 글자크기 변경하기
  • 그룹 선택자를 사용하여 테두리 적용하기
  • 자손, 후손 선택자를 사용하여 배경색, 글자모양, 글자색상 적용하기
  • 구조 선택자를 사용하여, 홀수, 짝수번째 배경색 적용하기
  • 동위 선택자를 사용하여 글자 밑줄 긋기
  • 탐색 선택자를 사용하여 홀수 짝수번째 글자모양 변경하기
  • 속성 선택자를 사용하여 글자색, 진하게 변경하기
    // 2 제이쿼리 랩퍼
      $(document).ready(function(){
      $('#title01').css('color','#00f');    //아이디 선택자를 사용하여 글자색 변경하기
      $('.p01').css('background','#ff0');   //클래스 선택자를 사용하여 배경색 변경하기
      $('body').css('font-size','14px');    //태그 선택자를 사용하여 글자크기 변경하기
      $('#title01, .p01').css('border','2px solid #f00'); //그룹 선택자를 사용하여 테두리 적용하기
      $('ul > li').css({'background':'#ccc','font-family':'맑은 고딕','color':'#00f'}); //자손, 후손 선택자를 사용하여 배경색, 글자모양, 글자색상 적용하기
      $('ul > li:nth-child(2n)').css('font-size','25px'); //구조 선택자를 사용하여, 짝수번째 배경색 적용하기
      $('ul > li:nth-child(2n+1)').css('background','pink'); //구조 선택자를 사용하여, 홀수번째 배경색 적용하기

      $('h2 + p').css('text-decoration','underline'); //동위 선택자를 사용하여 글자 밑줄 긋기

      $('ul > li:even').css('color','tomato');//탐색 선택자를 사용하여 홀수 짝수번째 글자색 변경하기
      $('ul > li:odd').css('color','red');//탐색 선택자를 사용하여 홀수 짝수번째 글자색 변경하기
      
      $('h2[id=title01]').css({'font-weight':'bolder', 'color':'red'});//속성 선택자를 사용하여 글자색, 진하게 변경하기
    });

제이쿼리에서 css 서식 적용하는 방법 3가지

    .css()메소드 사용법
    서식이 1개 : .css('속성','값');
    서식이 2개 이상 : .css('속성','값').css('속성','값').css('속성','값');
    서식이 2개 이상 : .css({'속성':'값','속성':'값','속성':'값'});
