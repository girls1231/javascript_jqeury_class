제이쿼리 소개와 문법

제이쿼리란 ? 자바스크립트의 생산성을 향상시켜주는 자바스크립트 라이브러리 언어이다.

자바스크립트 라이브러리란 ? 자주 사용하는 코드들을 재사용할 수 있는 형태로 가공하여 프로그래밍의 효율성을 높여주도록 만든 코드들의 집합을 말한다.

1. 제이 쿼리의 특징

  • 엘리먼트(html 태그요소)를 선택하는 강력한 기능을 제공(css 선택자보다 많음)
  • 선택된 엘리먼트를 제어할 수 있는 다양한 함수를 제공한다.
  • 자바스크립트의 자주사용하는 기능을 라이브러리화한 언어이다

2.제이쿼리 라이브러리 연결 방법

  • 직접 연결하기 - 제이쿼리 공식사이트에서 라이브러리 파일(.js)를 다운로드받아서 html 요소안에 직접 script 작성하여 연결
  • CDN방식으로 연결 - JQUERY.COM페이지에서 오른쪽'다운로드'클릭, 페이지로 이동하고, 아래페이지로 내려가면 'google CDN'메뉴 클릭하여 나온 페이지에서 1x, 2x, 3x 버전 선택하여 복사, html head 영역안에 붙여넣기 한다.

3. 제이쿼리 문법(작성하는 방법)

  • $('선택대상').메서드();
  • jquery('선택대상').메서드();

4. 제이쿼리에서 $의 역할

제이쿼리에서는 '$'기호를 맨 앞에 넣고 시작하는데, 이것은 '괄호()안에 넣을 대상을 선택하기 위함'이다.
예) $('h1'), $('window'), $('#id'), $('input[type='text']')...........

5. 제이쿼리에서 $기호 안에 들어가는 대상 3가지

  • $('선택자')
  • $('태그명')
  • $(자가스크립트 객체) - document, array, window.. etc

6. 제이쿼리 시작시 무조건 사용해야하는 wrapper 종류

  • $(document).ready(function(){ 실행할 내용}); - 기본버전
  • $(function(){ 실행할 내용}); - 축약버전
  • jQuery(function(){ 실행할 내용}); - 과거버전
  • $(function(){ 실행할 내용})(jQuery); - 과거버전
    $(document).ready(function(){ //html 문서가 모두 로딩이 끝나면 제이쿼리를 동작한다
    //자바스크립트 내용은 여기에 작성 가능
    //제이쿼리 내용은 반드시 여기에 작성!!!

    //자바스크립트로 제목서식 변경하기
    let title01;
    title01 = document.getElementById('title01');
    title01.style.color = '#00f';

    //제이쿼리로 제목 서식 변경하기
    $('h3').css('color','#f00');
    });

    $('pre').css({
      'background':'orange',
      'font-size':'16px' 
    });

    $('li:first-child').css('color','#00f'); //문서안에 li 태그를 선택하여 서식을 적용한다.
