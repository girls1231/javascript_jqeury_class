02_제이쿼리 객체조작_내용추가, 삭제 메서드

dom(문서객체모델)에서 문서안의 요소에 내용을 추가하거나 삭제할 수 있다

  • append(); - 사용자가 선택한 요소를 기준으로 안쪽으로 맨 뒤쪽에내용 추가
  • prepend(); - 사용자가 선택한 요소를 기준으로 안쪽으로 맨 앞쪽에 내용 추가
  • after(); - 사용자가 선택한 요소를 기준으로 바깥쪽으로 맨 뒤에 내용 추가
  • before(); - 사용자가 선택한 요소를 기준으로 바깥쪽으로 맨 앞에 내용 추가

insertAfter(), insertBefore() : A라는 요소 앞에 B를 배치할 수도 있고, B라는 요소뒤에 A를 배치할 수도 있다. after, before과 기능은 같지만 사용법이 다르다

예)
$('a').after('b'); - 요소의 바깥쪽으로 뒤에 b내용을 추가한다
$('b').insertAfter('a'); 요소의 바깥쪽으로 뒤에 b내용을 추가한다

문서 객체조작 - 내용을 삭제하기 위한 메서드 remove

사용자가 선택한 요소의 내용만 삭제하거나, 아예 태그요소를 삭제할 수 있다

  • remove(); 사용자가 선택한 요소를 제거
  • empty(); - 사용자가 선택한 요소의 내용만 제거
아래 remove 버튼을 클릭하면 이 박스요소가 제거됩니다
아래 empty 버튼을 클릭하면 이 박스요소안에 있는 내용만 제거됩니다
